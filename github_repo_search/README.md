# Github public repository search
https://gh-repo-search.herokuapp.com/

## Quick start
```
bundle
yarn
rails s
```

There are three env vars that can be edited but they are not needed for the application to work.
```
GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=
RESULTS_PER_PAGE=
```
